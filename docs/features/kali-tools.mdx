## Using Tools with aesc

### Natural Language Interface

```bash
# Simply describe what you want
aesc -c "scan example.com for open ports"
aesc -c "check if example.com has SQL injection"
aesc -c "brute force SSH on 192.168.1.100"
aesc -c "crack this hash: 5f4dcc3b5aa765d61d8327deb882cf99"
```

### Tool Discovery

```bash
# Ask aesc about tools
aesc -c "what tools can scan for XSS?"
aesc -c "how do I use sqlmap?"
aesc -c "show me wireless attack tools"
aesc -c "what's the best tool for subdomain enumeration?"
```

### Automated Workflows

aesc intelligently chains tools together:

```bash
# Reconnaissance workflow
aesc -c "perform full recon on example.com"
# → Runs: nmap, whatweb, sublist3r, gobuster, nikto

# Exploitation workflow
aesc -c "test example.com for common vulnerabilities"
# → Runs: nikto, sqlmap, xsser, wpscan (if WordPress)

# Password attack workflow
aesc -c "crack captured WPA handshake"
# → Runs: aircrack-ng with optimized wordlist
```
