## Welcome to aesc CLI

**aesc** is an AI-powered CLI agent that can autonomously perform security operations and penetration testing tasks. It provides a natural language interface to security tools like nmap, sqlmap, metasploit, and more.

## Why aesc?

<AccordionGroup>
  <Accordion title="🤖 Natural Language Interface">
    Describe tasks in plain English. No need to remember complex command syntax.

    ```bash
    aesc -c "scan 192.168.1.1 for open ports"
    aesc -c "analyze this web application for vulnerabilities"
    ```
  </Accordion>

  <Accordion title="🛡️ Built-in Approval System">
    Review every command before execution. Safety first.

    ```
    [aesc] Planning to execute: nmap -sV 192.168.1.1
    Approve? [y/N]:
    ```
  </Accordion>

  <Accordion title="🐧 Kali Linux Integration">
    Runs on a Kali Linux base with a curated security toolset pre-installed — the full Kali catalog is one `apt install` away.
  </Accordion>

  <Accordion title="🌐 LLM-Agnostic">
    Works with Ollama (local), OpenAI, Anthropic Claude, and more.
  </Accordion>
</AccordionGroup>

## Quick Example

<CodeGroup>

```bash Interactive Mode
aesc
> scan 192.168.1.1 for open ports
```

```bash Command Mode
aesc -c "scan 192.168.1.0/24 and identify services"
```

```bash Docker
docker run -it --rm \
  -e ANTHROPIC_API_KEY=your-key \
  ghcr.io/akaeli-aesc/aesc-cli:latest
```

</CodeGroup>
